# Tweeter

A Python tool that parses email content and generates tweetable content using OpenAI's API.

## Description

This project takes email content as input and uses OpenAI's language models to convert lengthy email text into concise, engaging tweets suitable for social media sharing.

## Features

- Email parsing and content extraction
- OpenAI integration for intelligent text summarization
- Tweet generation optimized for social media engagement
- Support for multiple tweet formats and styles

## Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/tweeter.git
cd tweeter
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Set up your OpenAI API key:
```bash
export OPENAI_API_KEY='your-api-key-here'
```
